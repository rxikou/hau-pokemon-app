# API Reference Folder

This folder is tracked in git so the team has one shared place for PHP API reference files.

## What to put here
- `get_monsters.php`
- `add_monster.php`
- `update_monster.php`
- `delete_monster.php`
- `update_player.php`
- any shared DB config examples (without real secrets)

## Notes
- Do not commit production secrets, passwords, or private keys.
- If this folder is only for reference, production deployment can still happen from your web server workflow.
